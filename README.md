# AntiTrain License

**A permissive open source license that prohibits AI training**

[![License: AntiTrain](https://img.shields.io/badge/License-AntiTrain%20v1.0-blue.svg)](LICENSE)

## What is AntiTrain?

AntiTrain is a software license that gives you all the freedoms of MIT/Apache licenses (use, modify, distribute commercially) while explicitly prohibiting the use of your code to train AI models.

## Why AntiTrain?

Many developers want to share their code openly but don't want it used to train large language models or other AI systems. Existing licenses don't address this:

- **MIT/Apache**: Too permissive - allows AI training
- **GPL**: Too restrictive - requires derivative works to be open source
- **Proprietary**: Too closed - limits legitimate use

AntiTrain fills the gap.

## What You Can Do

✅ Use the software commercially  
✅ Modify and create derivative works  
✅ Distribute and sell products built with it  
✅ Use privately in your company  
✅ Sublicense to others  

## What You Cannot Do

❌ Use the code to train AI/ML/LLM models  
❌ Include it in AI training datasets  
❌ Fine-tune models with it  

## Who Is This For?

- Developers who want to share code but protect it from AI training
- Projects that embrace human collaboration but not machine learning scraping
- Anyone seeking a middle ground between "totally open" and "totally closed"

## Important Notes

⚠️ **Not OSI-Approved**: This license intentionally restricts AI training, which violates OSI's "no discrimination against fields of endeavor" principle. This is a community license, not an OSI-approved open source license.

⚠️ **Legal Gray Area**: Courts haven't yet ruled on whether AI training constitutes "use" under existing licenses. AntiTrain makes the restriction explicit.

⚠️ **Enforcement**: Like all licenses, enforcement depends on copyright law in your jurisdiction. Consult a lawyer for specific legal advice.

ℹ️ **Not the Same as Elthian-NoTrain**: The [Elthian-NoTrain License](https://elthian.com/licensing/elthian-notrain-license-v1-0/) is a different license focused on allowing AI search while blocking training. AntiTrain is simpler and designed for software developers who want to block all AI training while keeping traditional open source freedoms.

## FAQ

### Is this actually enforceable?

The legal landscape around AI training and copyright is still evolving. This license makes your intent explicit. Whether it's enforceable depends on your jurisdiction and future court rulings.

### Can I use AntiTrain code in my project?

Yes! As long as you're not training AI models with it, you can use it commercially, modify it, and distribute products built with it.

### What counts as "AI training"?

The license prohibits using the code to train, fine-tune, or improve any AI/ML/LLM system. This includes adding it to training datasets, using it for model fine-tuning, or incorporating it into systems that generate training data.

### What if someone violates the license?

Like any license violation, you would pursue it through copyright law. The AntiTrain restriction is a condition of the copyright license.

### Can I dual-license my project?

Yes! You can offer your code under AntiTrain for general use, and separately offer commercial licenses that allow AI training for a fee.
